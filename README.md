### 2. 数据预处理
    一般来说，我们拿到的基础语料库可能是一些电影台词对话，或者是UBUNTU对话语料库(Ubuntu Dialog Corpus)，但基本上我们都要完成以下几大步骤
    1. 分词(tokenized)
    2. 将语料分成 Question和Answer部分
    3. 训练集和验证集划分
    4. 创建词汇表，对话转为向量形式
